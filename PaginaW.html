Nuestro proyecto

Este proyecto fue realizado como parte del trabajo final de la asignatura Tendencias en Desarrollo 3D por tres estudiantes universitarios, cuyo objetivo era recrear un entorno visual que fusionara la estética medieval con elementos naturalistas, utilizando Blender como principal herramienta de diseño. A lo largo de este proceso, cada uno de nosotros asumió roles específicos que nos permitieron abordar las diferentes fases del desarrollo, integrando varias plataformas y tecnologías que dieron vida a nuestra visión.

Información

Fase 1: Creación del Concepto y Modelado en Blender

Blender fue la plataforma central en la que creamos un ambiente de bosque con una casa de estilo medieval. Nuestra visión era representar un entorno naturalista, donde la arquitectura medieval coexistiera armoniosamente con la naturaleza, reflejando la tranquilidad y el misterio de esa época.

1. Modelado de la casa medieval: Comenzamos diseñando una casa de madera y piedra, inspirada en las estructuras típicas de la Edad Media. Utilizamos las herramientas de modelado de Blender para crear cada detalle de la casa, desde los pilares de madera hasta los techos inclinados de teja. Empleamos técnicas como el uso de extruding y loop cuts para definir la forma de las paredes y techos. Además, aplicamos texturas realistas de madera envejecida y piedra rústica utilizando nodos en el Shader Editor de Blender.

2. Creación del bosque: Paralelamente, se diseñó un entorno natural que rodeaba la casa. Utilizamos las herramientas de escultura de Blender para modelar un terreno irregular, añadiendo colinas, caminos y claros en el bosque. Mediante el uso de partículas, generamos vegetación variada, como árboles, arbustos y hierba que dieron vida al ambiente. Se aplicaron texturas PBR (Physically-Based Rendering) en las hojas, el suelo y las rocas, lo que ayudó a realzar el realismo de los materiales naturales.

3. Iluminación y detalles ambientales: La iluminación fue un factor clave para generar la atmósfera de misterio en el bosque. Se colocaron luces direccionales que simulaban la luz del sol filtrándose a través de los árboles, mientras que sombras suaves caían sobre el suelo. El motor de renderizado Cycles fue utilizado para obtener una iluminación realista y detallada, y ajustamos los parámetros globales para crear una atmósfera envolvente que capturara la calma de la naturaleza medieval.

Fase 2: Integración de GitHub para Colaboración en la Página Web

En cuanto al control de versiones, decidimos utilizar GitHub, pero su principal aplicación fue en la colaboración para el desarrollo de la página web que acompañaba el proyecto. GitHub nos permitió compartir y sincronizar el código que cada uno de nosotros desarrollaba en Visual Studio Code para la construcción del sitio web. Creamos múltiples repositorios donde subíamos y actualizábamos los archivos HTML, CSS y JavaScript necesarios para crear una página interactiva que mostrara tanto el proceso de creación del ambiente medieval como los renders finales. GitHub nos facilitó la colaboración, ya que cada miembro del equipo podía aportar al diseño del sitio, realizar cambios en el código y sugerir mejoras sin perder el control de las versiones. Usamos pull requests para revisar y aprobar los cambios, garantizando que todos los archivos estuvieran al día y correctamente integrados en la página principal.

Fase 3: Desarrollo Web con Visual Studio Code

Para complementar la presentación del proyecto, desarrollamos una página web informativa utilizando Visual Studio Code. Este editor nos permitió escribir y depurar el código HTML, CSS y JavaScript de manera eficiente. Nos enfocamos en crear una interfaz amigable que documentara el proceso de creación del ambiente en Blender, incorporando galerías de imágenes y un visor interactivo en 3D La página web fue diseñada para ser intuitiva y visualmente atractiva, reflejando el estilo naturalista-medieval del proyecto. Gracias a la estructura modular del código, pudimos implementar funciones interactivas como la exploración del entorno 3D y la navegación por las distintas etapas del proyecto.

Fase 4: Resultado Final e Impacto del Proyecto

El proyecto culminó con la creación de un entorno de bosque medieval detallado, complementado por una presentación web interactiva. La integración de Blender, Visual Studio Code y GitHub nos permitió trabajar de manera colaborativa, asegurando que cada aspecto del proyecto estuviera optimizado y documentado adecuadamente. El resultado final ofrece una experiencia inmersiva, donde los usuarios pueden explorar el bosque medieval y la casa de madera y piedra, rodeados de una naturaleza realista. Este proyecto no solo fue un desafío técnico, sino también una oportunidad para aplicar habilidades en diseño 3D y desarrollo web de manera integrada.
